HOW TO USE BIGBEN IRC COUNTER WIDGET

1. BigBen must be in your channel. Contact TheCrittaC on Rizon to request the bot.
2. $ChannelName variable must be changed to whatever the channel name is, including the pound sign (#)
3. irc_count.php can be included inline using the php method include('/path/to/irc_count.php');
